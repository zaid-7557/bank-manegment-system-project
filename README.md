# Bank Management System

##  Project Description
Bank Management System is a simple console-based application developed using C++.  
This project helps to manage bank accounts and perform basic banking operations.

## Features
- Create a new bank account
- Deposit money
- Withdraw money
- Check account balance
- Display account details

##  Technologies Used
- C++
- Object Oriented Programming (OOP)
- File Handling (if used)
- VS Code / Any C++ Compiler

##  Purpose of the Project
The main purpose of this project is to understand the concepts of:
- C++ Programming
- Classes and Objects
- Functions
- Basic Banking Operations Logic

##  How to Run the Program
1. Download or clone this repository.
2. Open the project in VS Code or any C++ compiler.
3. Compile the program.
4. Run the program and follow the instructions on the console.

##  Learning Outcome
This project helps beginners learn how to build real-world logic using C++ and Object-Oriented Programming.
